# GIT Commands


## Global settings -> One time on Computer
```bash
git config --global user.name "Your Name"
git config --global user.email "username@example.com"
```

## Initiate a new project (Reposotory) [REPO] -> One time on each project
```bash
git init
```

## Add file to stage
```bash
git add <FILE_path_name>
```

## Add all files to stage
```bash
git add .
```

## Commit your cahnges
```bash
git commit -m "<YOUR MESSAGE>"
```
